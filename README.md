# Project Title

Browser Evaluation

## Description

Firefox and Chrome simple browser using Python and Selenium.

## Getting Started

### Dependencies

* Python 3
* Selenium
* PIL
* pyscreenshot
* xlsxwriter
* platform

### Installing

* Clone the project
* Install Dependencies

### Executing program

* run 
```
python evaluate.py
```
